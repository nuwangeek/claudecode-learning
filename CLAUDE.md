# Project Instructions

## Documentation Sources

### LangGraph
Always use Context7 MCP server for any LangGraph-related questions, documentation, and code examples.

**Workflow:**
1. Use `mcp__context7__resolve-library-id` with `libraryName="langraph"`
2. Then use `mcp__context7__query-docs` with the resolved library ID

**Library IDs:**
- Python: `/langchain-ai/langgraph`
- JavaScript: `/langchain-ai/langgraphjs`
- Alternative sources available for different documentation needs
